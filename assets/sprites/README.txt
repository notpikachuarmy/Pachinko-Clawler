Recomendación futura:
- Tamaño de lienzo: 128x128 px para objetos pequeños.
- Alternativa de mayor detalle: 256x256 px.
- Fondo totalmente transparente.
- Objeto centrado.
- Evitar sombras externas semitransparentes si se usará la silueta para generar la hitbox.
